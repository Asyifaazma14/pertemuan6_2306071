#  pertemuan6_2306071

Project ini merupakan hasil praktikum **Pertemuan 6** pada mata kuliah Praktikum Pemrograman Mobile dengan menggunakan framework Flutter.

##  Tujuan

Project ini dibuat untuk mengimplementasikan:

* Form input menggunakan Flutter
* Validasi input (Form & Validator)
* Penggunaan TextEditingController
* Navigasi antar halaman (Navigator)
* Routing pada Flutter

---

##  Fitur Aplikasi

* Form Login dengan validasi
* Navigasi dari halaman Login ke Home
* Pengiriman data username ke halaman Home
* Tombol kembali ke halaman Login

---

##  Struktur Halaman

* **Login Page**

  * Input username
  * Validasi form
  * Tombol login

* **Home Page**

  * Menampilkan "berhasil login"
  * Tombol kembali

---

##  Cara Menjalankan Project

1. Clone repository:

   ```bash
   git clone https://github.com/USERNAME/pertemuan6_2306071.git
   ```

2. Masuk ke folder project:

   ```bash
   cd pertemuan6_2306071
   ```

3. Jalankan aplikasi:

   ```bash
   flutter run
   ```
